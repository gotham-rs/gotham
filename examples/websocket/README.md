# Websocket

Shows how to use Gotham with Tungstenite

## Running

From the `examples/websocket` directory:

```
Terminal 1:
  $ cargo run
    Finished dev [unoptimized + debuginfo] target(s) in 0.09s
     Running `.../gotham/target/debug/websocket`
Listening on http://127.0.0.1:7878/
```
